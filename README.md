## Codeigniter & MongoDB

- php -v 7.3
- codeigniter -v 4.1 
- mongoDB -v 4.2

The MongoDB dump stored in the /mongo_db folder as users.json file.
This is a CRUD project of users.

## Usage

You need to do the following steps:
- to install codeigniter 4
- run composer require mongodb/mongodb "^1.8"
